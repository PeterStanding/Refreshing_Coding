In order to run code in the command prompt

Move the Directory using --> D:
Move to the folder using cd Coding Practice\Java

Compile the Code by writing in CMD javac (file name).java
Run the Code by writing in CMD java (file name)

Compiled Code is in the main folder, and the completed code is within the java_code folder
